# Songs Of Westeros
A simple 2D action platformer based on the Game of Thrones' continent Westeros. As a Game of Thrones fan I wanted to create an experience 
based on the lore which I can share enjoy and also share with other similar fans. This project is currently in flux as i am still organising my 
thoughts and ideas however with time i will be able to update this Document with a more concrete version (Game Design Document).
